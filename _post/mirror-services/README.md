## mirrorservice.org

Sourceforge 是一些开源软件经常用到的网站，然而国内的网站一直不稳定。有时可以访问，但是一直无法下载。这是可通过 [mirrorservice.org](http://www.mirrorservice.org/sites/)，这个网站提供了众多网站的镜像，可以通过 web 或者 ftp 等方式来访问，代理的站点相当多。
