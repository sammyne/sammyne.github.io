## 关于 vuepress-theme-reco
- frontMatter 中的 `categories` 是必选项
- 正文节标题的最高层级为 2，即从 `##` 开始
- 常用图标参见 [图标](https://vuepress-theme-reco.recoluan.com/views/1.x/configJs.html#%E5%9B%BE%E6%A0%87)

## 关于 Github Actions
- 使用 `v{1,2,...}` 类型的字符串作为 `branches` 的值时会导致没有触发工作流
